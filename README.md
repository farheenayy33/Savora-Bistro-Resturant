## рџ“¦ Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/farheenayy33/Savora-Bistro-Resturant.
   ```

2. **Navigate to the project folder:**

     ```bash
   cd Savora-Bistro-Resturant
   ```

   ```bash
   npm run dev
   ```

   ```bash
   npm run build
   ```
